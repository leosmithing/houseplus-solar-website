## 🚀 快速开始

### 本地预览
```bash
# 进入项目目录
cd houseplus-solar

# 启动本地服务器
python3 -m http.server 8000

# 在浏览器中访问
http://localhost:8000
```

### 部署到 GitHub Pages

1. **创建 GitHub 仓库**
   - 访问 https://github.com/new
   - 仓库名称：`houseplus-solar-website`
   - 选择 Public（公开）
   - 点击 Create repository

2. **上传文件**
   - 点击 "uploading an existing file"
   - 上传本项目中的所有文件和文件夹
   - 或使用 Git 命令：
   ```bash
   git clone https://github.com/YOUR_USERNAME/houseplus-solar-website.git
   cd houseplus-solar-website
   # 复制所有文件到此目录
   git add .
   git commit -m "Initial commit: HousePlus Solar website"
   git push origin main
   ```

3. **启用 GitHub Pages**
   - 进入仓库 → Settings → Pages
   - 选择 "Deploy from a branch"
   - 选择 "main" 分支
   - 点击 Save
   - 等待几分钟，网站将在以下地址发布：
   ```
   https://YOUR_USERNAME.github.io/houseplus-solar-website
   ```
